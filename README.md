# Housing Price Prediction

## Project Overview
This project predicts California house prices using Machine Learning and Linear Regression.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Workflow
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Building
6. Model Evaluation

## Model Used
- Linear Regression

## Evaluation Metrics
- MAE
- MSE
- RMSE
- R² Score

## Dataset
California Housing Dataset from Scikit-learn
